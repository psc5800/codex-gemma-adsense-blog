# AdSense 승인 준비

## 필수 확인

- 소개, 개인정보처리방침, 문의 페이지가 실제로 연결되어 있어야 합니다.
- 승인 전에는 `ADSENSE_ENABLED=false`를 유지합니다.
- 글은 최소 10개 이상을 목표로 하되, 짧은 복제 글보다 검수된 글을 우선합니다.
- 광고 클릭 유도, 과장된 수익 보장, 출처 없는 수치를 피합니다.

## 30일 운영 리듬

| 기간 | 목표 |
| --- | --- |
| 1주차 | 필수 페이지, 사이트맵, 검색 콘솔 등록 |
| 2주차 | 핵심 주제 5개 글 발행 |
| 3주차 | 내부 링크, 제목 중복, 모바일 화면 점검 |
| 4주차 | 승인 신청 전 전체 감사와 수정 |

## 감사 명령

```powershell
npm run approve
```

이 명령은 글 수, 필수 페이지, 짧은 본문, 중복 제목, AdSense 설정을 확인합니다.
