# 개인정보처리방침

이 사이트는 정적 페이지로 운영되며, 자체 회원가입이나 댓글 기능을 제공하지 않습니다. 문의를 위해 사용자가 직접 이메일을 보내는 경우, 답변 목적에 필요한 범위에서만 정보를 사용합니다.

## 수집될 수 있는 정보

- 사용자가 문의 메일에 직접 적은 이름, 이메일 주소, 문의 내용
- GitHub Pages, 검색 엔진, 광고 서비스 등 외부 서비스가 제공하는 집계 통계

## 사용 목적

- 문의 응대
- 사이트 품질 개선
- 보안과 오남용 방지

## 보관과 삭제

문의 기록은 필요한 기간 동안만 보관하며, 삭제 요청이 있으면 합리적인 기간 안에 처리합니다.

## 외부 서비스

향후 Google AdSense 또는 검색 콘솔을 연결할 수 있습니다. 해당 서비스는 각 제공자의 정책에 따라 쿠키나 집계 데이터를 처리할 수 있습니다.
